    Request 5: Lecture Alignment¶

    SYSU Advanced Computer Networks ▶ CMU 15-641 Networking and the Internet

    Coverage: 79.55% ≥ 75%  🟩 Full 11 · 🟩 Mostly 7 · 🟨 Partial 2 · ⬜ Conceptual 1 · ⬛ Indirect 1 (n=22)

    CMU lecture names below are taken verbatim from the official CMU 15-441/641 Fall 2024 schedule; each lecture title is itself a clickable link to the corresponding slide deck on Prof. Steenkiste's Google Drive. SYSU evidence cites the four slide decks (Nn s.X = deck n, slide X, click to jump to that slide) and the final group project (an applied network-security research artifact on Ethereum P2P / cryptographic protocols).

    Notes on coverage¶

    • The 18 fully or largely covered lectures span the complete CMU 15-641 core: performance, OSI / IP layering, routing (DV + LS + BGP + SDN), QoS, DNS, HTTP / CDN / VoD, the entire security stack (crypto, TLS, IPsec, firewalls / NFV), and all wireless / cellular content.

    • The 2 partial lectures cover topics that SYSU introduces but does not derive in depth (Ethernet's CSMA/CD, intra-DC Clos topology).

    • Only TCP windowing (L10) and advanced congestion control (L11) are genuinely thin on the SYSU side, as expected for a graduate-level course that re-frames classical mechanisms via SDN / traffic-engineering perspectives.

    • The final group project is research-grade applied network security (Ethereum P2P, ECDH key exchange, ECDSA derivation, applied routing-attack analysis) that exceeds the scope of a typical 15-641 homework, providing strong evidence for the cryptography and routing-security rows.
