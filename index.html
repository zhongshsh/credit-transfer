    Credit Transfer Portfolio¶

    Applicant: Shanshan Zhong (SYSU Master, ID 22214624) → CMU PhD program (szhong2)

    This site documents 6 credit-transfer requests mapping graduate-level courses I took at Sun Yat-sen University (SYSU) to courses at Carnegie Mellon University (CMU). Each request provides exactly the three parts:

    1. a public URL to the CMU course syllabus
    2. a lecture-by-lecture mapping with a quantified ≥ 75 % coverage check
    3. a homework comparison between the two courses

    Request overview¶

    All SYSU courses are 3-credit graduate courses (54 class periods of 45 min each, over an 18-week semester). Grades are on a 0–100 scale; passing = 60.

    # SYSU Course CMU Course Grade
    1 Advanced Computer Networks 15-641 Networking and the Internet 92
    2 Pattern Recognition + Computational Visual Media 11-777 Multimodal Machine Learning 88 / 90
    3 Stochastic Processes 15-857 Analytical Performance Modeling 85
    4 Advanced Algorithms + Machine Learning 10-601 Introduction to Machine Learning 88 / 85
    5 Advanced Algorithms and Programming Techniques 15-451/651 Algorithm Design and Analysis 88
    6 Machine Learning 10-703 Deep Reinforcement Learning and Control 85

    Requests 5 & 6 are deprecated

    • Request 5 (Algorithms → 15-651) is superseded by Request 4 (which combines Algorithms + ML into a two-for-one mapping against CMU 10-601).
    • Request 6 (ML → 10-703) scored below 75 % and is kept for reference only. Neither should be submitted.

    Changes from v1 (in response to Prof. Jamie Callan's 23 Jan 2026 review)

    The original (v1) credit-transfer petition mapped courses to CMU targets that Prof. Callan correctly identified as poor fits. This v2 dossier re-targets those requests and reorganises the numbering after deeper lecture-by-lecture analysis:

    Re-targeted CMU courses:

    v1 target v2 target Reason for change
    15-750 Algorithms in the Real World 10-601 Introduction to Machine Learning (Request 4, two-for-one: Algo + ML) 15-750 is CMU's advanced electives course (SVD, cache-oblivious algorithms, etc.) — a poor match. The SYSU Algorithms course alone mapped to 15-451/651 at only 66.67 %, below the 75 % threshold. By combining it with the SYSU ML course, the joint coverage of 10-601 reaches 81.48 %.
    10-701 Introduction to Machine Learning 10-601 Introduction to Machine Learning (same Request 4) v1 mapped SYSU ML alone to 10-701; v2 initially re-targeted to 10-703 (Deep RL) but that scored only 59.62 %. The ML course is now combined with Algorithms in the two-for-one Request 4 against 10-601.

    Coverage at a glance¶

    Scoring model. Each row in a Lecture Alignment page corresponds to one CMU lecture. A lecture counts as covered iff SYSU has actually taught its content:

    Badge Score Meaning
    Fully Covered 1.00 The lecture is taught at SYSU
    Mostly Covered 0.75 Main content taught; minor details may be missing
    Partially Covered 0.50 Roughly half of the lecture's material is taught
    Conceptually Related 0.25 Only conceptually related; not actually taught
    Indirectly Supported 0.00 Only theoretical prerequisites are present
    \[ \text{Coverage} \;=\; \frac{\sum_{\text{lecture}} \text{score}}{N_{\text{CMU lectures}}} \qquad \text{Pass threshold} = 75\%. \]

    scripts/compute_coverage.py parses the badges and rewrites the bars below.
