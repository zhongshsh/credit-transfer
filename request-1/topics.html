    Request 1: Lecture Alignment¶

    SYSU Advanced Algorithms and Programming Techniques ▶ CMU 15-451/651 Algorithm Design and Analysis

    How to read this page¶

    The official 15-451/651 Spring 2025 schedule contains 25 lectures (recitations omitted here — they reinforce the same lecture material). SYSU's single graduate course covers the algorithm-design core (Lower Bounds, Dynamic Programming, Network Flows, Linear Programming, Online Learning, and Approximation); the auxiliary topics (Hashing, Amortized Data Structures, Range Queries, Streaming, FFT, Gradient Descent) are taught at SYSU through specific undergraduate prerequisite courses which are listed in §2 with exact transcript citations (course name, hours, credits, grade). Computational Geometry (L21, L22) is a genuine gap — no SYSU course covers it — so those rows remain in §1 and are scored honestly.

    Three complementary views:

    1. §1 Per-lecture mapping (counted) — 15 lectures (the 13 design-core lectures plus the 2 Computational-Geometry lectures where SYSU coverage is genuinely absent), strictly scored against the official 15-451/651 schedule.
    2. §2 Auxiliary modules covered by SYSU prerequisites (excluded from scoring) — 10 lectures whose material is taught in SYSU's data-structures, big-data analysis, mathematical-modeling, and ML-fundamentals undergraduate courses, each cited by transcript line.
    3. §3 Additional content beyond 15-451/651 — substantial SYSU material (NP-completeness, approximation, randomized algorithms, intractability strategies) that CMU students obtain by enrolling in additional courses (15-455 / 15-854 / 15-859M / 15-459).

    3. Additional content beyond 15-451/651¶

    Four substantial SYSU modules have no counterpart in 15-451/651 and therefore do not appear in the lecture-by-lecture table. At CMU, these are each delivered by a separate course. SYSU packages them into the same single course, so a SYSU student receives in one course what a CMU student would obtain by enrolling in 15-451/651 plus the course shown below.

    SYSU module CMU equivalent course SYSU evidence
    NP-Completeness — Cook–Levin, classical reductions, P / NP / coNP, vertex-cover, 3SAT, Hamiltonian-cycle, TSP, ILP-hardness 15-455 Undergraduate Complexity Theory (entire course) S5 entire deck (84 pp)
    Approximation Algorithms — vertex cover, scheduling, TSP, knapsack PTAS / FPTAS, set cover, subset-sum FPTAS 15-854 Approximation Algorithms (Anupam Gupta, Fall 2021 archive) — also one lecture in 15-850 (L25) S6 entire deck (65 pp) — far deeper than 15-451 L19
    Randomized Algorithms — probability theory, randomized QuickSort, Karger's Min-Cut, Las Vegas vs Monte Carlo, Markov chains, 2SAT random walk, Chernoff 15-859M Randomized Algorithms (Anupam Gupta, Spring 2011 archive — the canonical CMU offering) S7 entire deck (138 pp)
    Strategies for Intractable Problems — 2SAT poly-time, HornSAT, fixed-parameter algorithms, exponential-time improvements, simulated annealing, average-case analysis, hard-instance generation, survey-propagation, quantum algorithms intro No single CMU equivalent. Quantum-intro maps to 15-459 Undergraduate Quantum Computation; FPT / exponential-time / SA / survey-propagation are typically taught only as 15-859-prefix special-topics seminars. S8 entire deck (104 pp)

    Curriculum-organization comparison. 15-451/651 covers algorithm design and analysis as a single course. SYSU packages the same core material with the additional content shown above (NP-completeness, approximation, randomized algorithms, intractability strategies) — material that, at CMU, is delivered through 3–4 separate courses (15-455, 15-854, 15-859M, 15-459). The strict per-session table in §1 covers SYSU's algorithm-design core; the auxiliary §2 modules are taught at SYSU through prerequisite / separate courses, and the §3 modules are taught at CMU through additional courses SYSU consolidates into this single offering.
