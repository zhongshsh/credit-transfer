    Request 4: Lecture Alignment¶

    SYSU Advanced Algorithms + Machine Learning ▶ CMU 10-601 Introduction to Machine Learning

    • CMU — 10-601 Introduction to Machine Learning

      Focus. A broad graduate-level introduction to machine learning covering classification, regression, neural networks, deep learning, learning theory, reinforcement learning, ensemble methods, unsupervised learning, and generative models. Emphasises both the mathematical foundations (optimisation, probability, PAC learning) and practical implementation through programming assignments.

      Topics: Decision Trees, k-NN, Perceptron, Linear / Logistic Regression, Optimisation (SGD), Neural Networks, Backpropagation, CNNs, RNNs, Transformers, Pre-training / Fine-tuning, PAC Learning, Reinforcement Learning (MDPs, Q-Learning, Deep RL), Ensemble Methods (Boosting, Bagging), K-Means, PCA, Recommender Systems, Generative Models, Bayesian Methods.

      Objectives.

      1. Formulate real-world problems as supervised, unsupervised, or sequential decision tasks.
      2. Implement and analyse classical and deep-learning algorithms from scratch.
      3. Reason about generalisation via PAC learning and bias-variance decomposition.
      4. Apply reinforcement-learning and ensemble methods.
      5. Connect theoretical guarantees to practical experimental design.

    • SYSU — Advanced Algorithms and Programming Techniques (Algo) + Machine Learning (ML)

      Focus.

      • Algo. Graduate-level algorithm design and analysis: divide-and-conquer, DP, greedy, LP / duality / Simplex, network flows, NP-completeness, approximation algorithms, randomised algorithms, intractability strategies.
      • ML. Graduate-level machine learning: Bayesian classifiers, MaxEnt models, boosting, MCMC, pre-trained models, prompt learning, causal inference, RL (MDP, policy gradient, PPO), model-based RL, imitation learning.
      • Combined. Provides both the optimisation / algorithmic backbone (LP, gradient methods, complexity) and the machine-learning content (classification, deep learning, RL, ensemble, generative, Bayesian) that 10-601 requires.

      Objectives.

      • Algo. Design efficient algorithms; analyse complexity; understand NP-completeness; implement and evaluate algorithms on real data.
      • ML. Understand probabilistic / statistical ML principles; apply discriminative / generative models; implement RL algorithms; explain pre-trained and prompt-based paradigms; apply causal reasoning.
      • Combined. Master the full pipeline from problem formulation → algorithm / model design → optimisation → implementation → evaluation.

    Genuine remaining gaps (kept honest):

    • Decision Trees (L3) and Decision Tree Hyperparameters (L4) — classic introductory ML classifiers not explicitly taught in either SYSU course. Both courses assume these as prerequisites and focus on more advanced methods.
    • k-Nearest Neighbors (L5) — kNN classifier with k-selection not a standalone topic; only the distance-metric and model-selection aspects are partially covered.
    • Societal Impacts (L14) — non-technical lecture with no SYSU counterpart.
    • PAC Learning (L15) — the formal sample-complexity framework is not derived, though the Algo course's complexity theory provides a related mindset.
    • Coding Agents / Significance Testing (L26) — new Spring 2026 special topic on LLM-based coding agents and statistical significance testing; no SYSU counterpart.

    Beyond 10-601 — extra SYSU material¶

    NP-Completeness — Cook–Levin, reductions, P / NP / coNP 15-455 Complexity Theory S5 entire deck (84 pp)
    Approximation Algorithms — vertex cover, scheduling, TSP, knapsack PTAS/FPTAS 15-854 Approximation Algorithms S6 entire deck (65 pp)
    Randomised Algorithms — randomised QuickSort, Karger's Min-Cut, Chernoff 15-859M Randomised Algorithms S7 entire deck (138 pp)
    Causal Inference — backdoor / frontdoor / counterfactual (4 lectures) No 10-601 equivalent; taught in CMU-MLD research seminars M8–M11 (4 decks) + A2 Report
    Deep RL — PPO, Actor-Critic, GAE, DQN variants, Model-based RL, Imitation Learning 10-703 Deep RL M12–M15 (4 decks) + A3 Code
    Prompt Learning & In-context Learning — prefix / cloze prompts, prompt tuning Advanced NLP electives M7 entire deck

    Curriculum-organisation comparison. 10-601 is CMU's single introductory ML course for Master's students. The two SYSU courses combined cover the same introductory ML topics plus six additional course-equivalents (convex optimisation, algorithm design, complexity theory, approximation, randomised algorithms, deep RL). The strict per-lecture table above scores only the 10-601 content; the "Beyond" table documents the substantial surplus.
