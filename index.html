    Credit Transfer Portfolio¶

    Applicant: Shanshan Zhong (SYSU Master, ID 22214624) → CMU PhD program (szhong2)

    This site documents 5 credit-transfer requests mapping graduate-level courses I took at Sun Yat-sen University (SYSU) to courses at Carnegie Mellon University (CMU). Each request provides exactly the three parts:

    1. a public URL to the CMU course syllabus
    2. a lecture-by-lecture mapping with a quantified ≥ 75 % coverage check
    3. a homework comparison between the two courses

    Request overview¶

    All SYSU courses are 3-credit graduate courses (54 class periods of 45 min each, over an 18-week semester). Grades are on a 0–100 scale; passing = 60.

    # SYSU Course CMU Course Grade
    1 Advanced Algorithms and Programming Techniques 15-451/651 Algorithm Design and Analysis 88
    2 Stochastic Processes 15-857 Analytical Performance Modeling 85
    3 Machine Learning 10-703 Deep Reinforcement Learning and Control 85
    4 Pattern Recognition + Computational Visual Media 11-777 Multimodal Machine Learning 88 / 90
    5 Advanced Computer Networks 15-641 Networking and the Internet 92

    Why Request 4 maps two SYSU courses to one CMU course

    SYSU Pattern Recognition and Computational Visual Media were taught as a consecutive multimodal-learning sequence. PR provides the multimodal foundation (Bayesian, NN, HMM, sequence/vision/language). CVM provides visual–language integration (geometry, video understanding, sketch). Together they cover the six MMML challenges (representation, alignment, reasoning, generation, transference, quantification) that 11-777 is built around.

    Changes from v1 (in response to Prof. Jamie Callan's 23 Jan 2026 review)

    The original (v1) credit-transfer petition mapped two requests to CMU courses that Prof. Callan correctly identified as poor fits. After re-reading the referenced CMU syllabuses page-by-page against my SYSU slides, this v2 dossier re-targets two requests to courses whose published lecture content more matches what I studied:

    v1 target v2 target Reason for change
    15-750 Algorithms in the Real World 15-451 / 15-651 Algorithm Design and Analysis 15-750 is CMU's advanced electives course (SVD, MST cache-oblivious algorithms, etc.); the SYSU course is a classical algorithm-design course — a textbook match for 15-451/651, not 15-750.
    10-701 Introduction to Machine Learning 10-703 Deep Reinforcement Learning and Control The SYSU course is a 3-instructor breadth survey whose Module 3 (M12–M15) + Assignment 3 PPO implementation are squarely in deep-RL territory; 10-703 is the closest CMU course where the PPO + GAE + Actor-Critic + MBRL coverage I have can be evaluated against published lectures.

    Coverage at a glance¶

    Scoring model. Each row in a Lecture Alignment page corresponds to one CMU lecture. A lecture counts as covered iff SYSU has actually taught its content:

    Badge Score Meaning
    Fully Covered 1.00 The lecture is taught at SYSU
    Mostly Covered 0.75 Main content taught; minor details may be missing
    Partially Covered 0.50 Roughly half of the lecture's material is taught
    Conceptually Related 0.25 Only conceptually related; not actually taught
    Indirectly Supported 0.00 Only theoretical prerequisites are present
    \[ \text{Coverage} \;=\; \frac{\sum_{\text{lecture}} \text{score}}{N_{\text{CMU lectures}}} \qquad \text{Pass threshold} = 75\%. \]

    scripts/compute_coverage.py parses the badges and rewrites the bars below.

    Prior correspondence with Prof. Callan¶

    This dossier was prepared in response to the initial decision of 23 January 2026. The exchange below records what was previously raised and the points I clarified in reply; this site provides the slide- and homework-level evidence that the clarifications were based on.

    📨 Initial decision — Prof. Jamie Callan, 23 Jan 2026¶

    Hi Shanshan,

    I reviewed your request for transfer credit four courses. Decisions are below.

    1. SYSU Advanced Algorithms and Programming Techniques → CMU 15-750 Algorithms in the Real World There is very little overlap between these courses. For example, 15-750 covers minimum spanning trees, dynamic programming, union-find, shortest paths algorithms, dynamic programming, hashing, SVD, and other topics not covered by SYSU Advanced Algorithms and Programming Techniques.

    2. SYSU Stochastic Processes → CMU 15-857 Analytical Performance Modeling and Design of Computer Systems There is very little overlap between these course. The CMU course is focused on stochastic processes for performance modeling of datacenter tasks. The SYSU course is a more general coverage of stochastic processes. For example, 15-857 covers ergodicity theory, capacity provisioning, Jackson networks, busy periods, scheduling, and other topics not covered by Stochastic Processes.

    3. SYSU Machine Learning → CMU 10-701 Machine Learning. There is very little overlap between these courses. For example, 10-701 covers decision trees, kNN, linear regression, regularization, backpropagation, CNNs, RNNs, clustering, learning theory and other topics not covered by SYSU Machine Learning.

    I am sorry, but I am not able to approve these transfer credit requests. I will respond to your requests for the other 2 courses soon.

    Best wishes, Jamie

    ✉️ My reply — Shanshan Zhong¶

    Hi Prof. Callan,

    Thank you very much. I fully understand your decision.

    I would, however, like to respectfully clarify a few points regarding the perceived overlap between the courses, in case this information is helpful for reconsideration.

    1. SYSU Advanced Algorithms and Programming Techniques → CMU 15-750 SYSU Advanced Algorithms and Programming Techniques explicitly covers dynamic programming (Slide 1) and shortest-path algorithms (Slide 3).

    2. SYSU Stochastic Processes → CMU 15-857 SYSU Stochastic Processes explicitly covers the ergodicity-related theory of Markov chains and steady-state behavior (SP06 – Finite-state Markov Chains, SP07 – Stochastic Matrices (Steady-State), SP08 – CTMC), and partially scheduling content (SP14, SP15 — FCFS as the service discipline for G/G/1 and G/G/m models).

    Best, Shanshan
