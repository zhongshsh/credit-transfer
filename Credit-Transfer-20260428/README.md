### `1-Advanced Algorithms and Programming Techniques`
- **Slices:** 9 lecture slides (Linear Programming, Network Flow, NP-Completeness, Approximation, Randomized Algorithms, etc.).  
- **Assignment:** Includes assignment description, written report (DOCX + PDF), and Python implementation (`main.py`, `dinic.py`, with image outputs).  
- **Textbook:** Algorithm Design and Analysis (2nd Edition).  

---

### `2-Stochastic Process`
- **Slices:** 15 lecture slides covering Poisson processes, Markov chains, MDPs, dynamic programming, and reinforcement learning.  
- **Assignments:** 12 written homework sets and one programming assignment.  
- **Exams:** Final exam (blank and answered versions).  
- **Textbooks:** 5 key references including *Gallager (Stochastic Processes)*, *Bertsekas (Dynamic Programming)*, and *Mitzenmacher & Upfal (Probability and Computing).*  

---

### `3-Machine Learning`
- **Slices:** 15 lecture slides on Bayesian classifiers, logistic regression, boosting, MCMC, causal inference (backdoor/frontdoor/counterfactuals), and reinforcement learning.  
- **Assignments:**  
  - `Assignment Discription.txt`: Assignment description
  - `Assignment2-Report`: Written report (Chinese and English versions).  
  - `Assignment3`: Reinforcement learning coding project (Python code + presentation slides).

### `4-Computational Visual Media`

- **Assignments:**
