    Request 4: Syllabuses¶

    Two SYSU courses → one CMU course

    SYSU Pattern Recognition (PR) and Computational Visual Media (CVM) were designed as a consecutive sequence forming a single graduate track on multimodal learning. PR provides the multimodal foundation; CVM focuses on visual–language integration and multimodal generation. Together they cover the six MMML challenges (representation, alignment, reasoning, generation, transference, quantification) that CMU 11-777 is built around. Total credit: 3 + 3 = 6 credits / 108 contact hours.
