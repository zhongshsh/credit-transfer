    Request 4: Syllabuses¶

    Two SYSU courses → one CMU course

    SYSU Advanced Algorithms and Programming Techniques (Algo) and Machine Learning (ML) are combined to map against CMU 10-601 Introduction to Machine Learning. The Algo course provides the optimisation and algorithmic foundations (LP, gradient methods, complexity theory) while the ML course provides the learning algorithms and applications (Bayesian methods, deep learning, RL, generative models). Together they cover the breadth of 10-601 and go substantially beyond it. Total credit: 3 + 3 = 6 credits / 108 contact hours.

    SYSU — Machine Learning (ML)¶

    Field Value
    Institution Sun Yat-sen University, School of Computer Science and Engineering
    Instructors Prof. Yanghui Rao (raoyangh@mail.sysu.edu.cn)
    Prof. Hanjiang Lai (laihanj3@mail.sysu.edu.cn)
    Prof. Ning Tan (tann5@mail.sysu.edu.cn)
    Credit 3 credits (54 class periods of 45 min each, over 18 weeks)
    Textbooks • Zhihua Zhou, Machine Learning, Tsinghua Univ. Press, 2016
    • Christopher M. Bishop, Pattern Recognition and Machine Learning, Springer, 2006
    Grading 40 % participation · 60 % testing / reports
    Final grade 85 / 100
    Slices M1 — Introduction
    M2 — Bayesian Classifier
    M3 — Maximum Entropy Model
    M4 — Boosting Method
    M5 — Markov Chain Monte Carlo (MCMC)
    M6 — Big data, Big model
    M7 — Prompt-based learning
    M8 — Introduction to Causal Inference
    M9–M11 — Causal Models I–III
    M12 — Reinforcement Learning 1
    M13 — Reinforcement Learning 2 — MDP
    M14 — Model-based RL
    M15 — Imitation Learning

    Course structure (3 modules):

    Module Instructor Topics SYSU Lectures
    1 — Probabilistic ML Prof. Yanghui Rao Bayesian, MaxEnt, Boosting, MCMC M1–M5

    3 — Reinforcement & Imitation Learning Prof. Ning Tan RL foundations, MDP, Model-based RL, Imitation Learning M12–M15
