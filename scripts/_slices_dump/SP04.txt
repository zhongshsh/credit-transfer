Introduction Arrival Processes
Arrival Processes
An arrival process is a sequence of increasing rv.s 0 < S1 < S2 < . . .
where Si < Si+1 means that X1 = S1 and Xi = Si − Si−1 for i > 1 are
positive rv.s, FXi (0) = 0 . The differences Xi are call interarrival times and
the Si are called arrival epochs .
For each t > 0, N(t) is the number of arrivals in (0, t]. We call
{N(t) : t > 0} an arrival counting process .

Definition and properties of a Poission process Memoryless Property
Memoryless Property
The remarkable simplicity of Poisson processes is closely related to the
memoryless property:
Definition 3
A rv. X is memoryless if X is positive and, for all real t > 0 and x > 0,
P{X > t + x} = P{X > t}P{X > x}.
Since the interarrival interval for a Poisson process is exponential, i.e.,
P{X > x} = exp(−λx) for x ⩾ 0,
exp(−λ(x + t)) = exp(−λt) exp(−λx).
An arbitrary rv. X is memoryless iff it is exponential.

Stationary increment property
Definition 4
A counting process {N(t) : t > 0} has the stationary increment property if
N(t′) − N(t) has the same CDF as N(t′ − t) for every t′ > t > 0.
Let
eN(t, t′) = N(t′) − N(t).
Stationary increments means that eN(t, t′) has the same distribution as
N(t′ − t).
Thus, the distribution of the number of arrivals in an interval depends on
the size of the interval but NOT the starting point.
