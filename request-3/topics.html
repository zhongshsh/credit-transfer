    Beyond the 10-703 syllabus — extra SYSU material¶

    A pure CMU-side mapping understates the SYSU course. SYSU Machine Learning is a broad three-module survey (probabilistic ML + modern DL & causality + RL), of which only Module 3 maps onto 10-703. The following SYSU material is not in 10-703 at all but demonstrates graduate-level breadth in adjacent areas:

    SYSU topic Slides / Artifact Why it strengthens the equivalency case
    Bayesian classifiers + naive Bayes M2 — Bayesian Classifier Foundational probabilistic ML — required prerequisite for any RL course but not taught in 10-703 itself
    Maximum-entropy model (Lagrangian dual, conditional entropy H(Y|X)) M3 — Maximum Entropy Model The mathematical machinery underlying MaxEnt RL / Soft Actor-Critic policy regularisation
    Boosting (AdaBoost, gradient boosting) M4 — Boosting Method Ensemble learning — used in offline-RL Q-ensemble methods but not taught in 10-703
    MCMC (Metropolis-Hastings, Gibbs, accept-reject sampling) M5 — MCMC Foundation for Bayesian / posterior-sampling RL exploration; also underlies MCMC-based model-based RL
    Self-supervised visual learning (MoCo, MAE, BEiT, ViT) M6 s40–60 Already credited in main mapping (10-703 SSL lecture) but the SYSU coverage is broader
    Causal inference — backdoor / frontdoor / counterfactual M8 – M11 (4 lectures, ~20 % of course) + A2 Causal-Inference Report (EN) Entirely absent from 10-703; an active CMU-MLD research area (Zhang, Schölkopf collaborations)

    Why coverage is below 75% — honest assessment¶

    The RL core is solidly covered. The 14 lectures rated Fully or Mostly include the entire deep-RL core (MDPs · VI / PI · TD / MC learning · function approximation · DQN family · PG / REINFORCE / A-C · PPO with paper-grade implementation · BC · MBRL · SSL · sensory→latent planning · visual IL · language→robot control). The PPO assignment (paper read + Clipped Surrogate + Adaptive KL Penalty + GAE + Actor-Critic on a non-trivial NLP task) is comparable in scope to a full 10-703 HW.

    The remaining gaps are real, not framing. Multi-armed Bandits (lecture 2 of 10-703, foundational) and Evolutionary Methods are genuinely standard graduate-RL content that SYSU does not cover. A second tier of nine lectures is honestly scored as Conceptual rather than Partial / Indirect because SYSU's M12 "State-of-the-art Algorithms" pages (p.60–62) list the canonical methods by name only — DDPG / TD3 / SAC (L10), MOReL / MOPO / MBOP / LOMPO (L20–L21), HER (L14) — without the deterministic-PG / re-parametrization / conservative-Q / distribution-shift derivations that the corresponding 10-703 lectures spend a full hour on. Likewise M14 p.15 names ExIt / AlphaZero (L15) inside the MBRL taxonomy without self-play or PUCT details; M12 p.54 states the exploration–exploitation tradeoff (L19) without UCB / count-based / RND / ICM; M12 p.5–9 names sim→real transfer + Progressive Net / PathNet (L22) without domain randomization or system identification; M14 p.57's robot-arm pipeline uses Gaussian-Process dynamics (L26, PILCO-era) rather than Diffusion Policy / NeRF / point-cloud representations; and M15 p.33 draws the GAN-IRL analogy (L13 / L14) without the GAIL discriminator objective. Together these honest down-ratings are the structural reason for the sub-75% score.
